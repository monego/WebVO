Modelo Padrão

Publicado Por: INPE
Última Modificação: Jul 19, 2018 10h38

Instruções

Para utilizar o Modelo Padrão de acordo com as orientações do INPE + Governo Federal, foi disponibilizado um modelo de arquivos para utilização no ambiente dos Servidores WEB da COCTI.


Configuração

O modelo foi estruturado para ser utilizado em Servidor Apache PHP e utiliza css, jquery e javascript para formatação do layout.

Para que o modelo funcione em um Servidor próprio, é necessário utilizar a configuração acima e seguir os seguintes passos:

  1. Realize o Download do Modelo através do link de menu "Download Modelo";
  2. Descompacte o conteúdo do arquivo na raiz do Servidor;
  3. Acesse o seu servidor (http://servidor/) e verifique se o conteúdo é exibido igual ao link: http://portalnovo.sir.inpe.br/modelo/;


Coordenação / Centro / Laboratórios

As seguintes alterações devem ser realizadas para a configuração do site.

  1. Alterar os arquivos index.php, topo.php e menu.php para que títulos indiquem o site correto.

    topo.php:

    • <img src="/img/logotipoInpe.png" alt="Logotipo do INPE"> para:
      <span class="portal-title">Nome</span>

    menu.php:
    Todos os itens de menu devem possuir caminho absoluto:

    • <a href="/" title="Acesse Sobre">

    index.php:

    • <title>INPE/Modelo Padrão</title> para:
      <title>INPE/Nome</title>
  2. Para que o RASTRO funcione, é necessário acrescentar um arquivo !.txt dentro de cada pasta criada. O conteúdo desse arquivo deve conter o 'nome da categoria do site' + 'ponto e vírgula' + 'página de link'. Ex: 'Nome;index.php' ;
  3. Caso o modelo seja para uma subpasta, deve-se copiar os arquivos rodape.php, topo.php, index.php e menu.php para dentro da nova pasta referente ao site e utilizar o link absoluto para os includes que ficaram na raiz.


Serviços e Programas

Além das alterações realizadas de acordo com o modelo de Coordenação / Centro / Laboratórios, as seguintes alterações devem ser realizadas para a mudança da cor do site e alteração do rodapé.

  1. Alterar o arquivo index.php para a inclusão dos arquivos '.css'.

    <link media="all" href="/css/css-servico-inpe.css" rel="stylesheet" />

    <link media="all" href="/css/css-branco-inpe.css" rel="stylesheet" />
