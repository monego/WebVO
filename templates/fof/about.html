Friends-of-Friends

Friends-of-Friends (FoF) is an algorithm for astronomical objects classification to identify which objects are in gravitational interaction. For this, it uses data from cosmological simulations.

The classification is applied in data from N-body simulations considering an percolation radius defined by the user. The basic idea of this algorithm is: consider a sphere of radius R around each particle of a given set. If within that sphere exist other particles, they will be considered as belonging to the same halo and be called "friends". Then, we take a sphere around each friend and the procedure continues using the rule "any friend of my friend is my friend". The procedure stops when no new friend can be added to the group.
