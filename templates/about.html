About

This Virtual Observatory allows remote execution of some applications in Astronomy.

The portal offers parallel versions of the Friends-of-Friends algorithm for astronomical objects classification, and algorithms for image restoration using wavelet filtering. The users can perform as many executions as they wants, and at the end of each they can download the output file and a log file.

This project was originaly developed by Otávio Migliavacca Madalosso in 2015 as a conclusion work in his graduation in Computer Science ate Federal University of Santa Maria (UFSM). Currently, this is continued by Ana Solórzano and Vinícius Monego.
